# Madeserra Madeireira - Mini Site Institucional

Projeto desenvolvido para a disciplina de Desenvolvimento Fullstack – Engenharia da Computação – SENAI CIMATEC.

## 📌 Sobre o Projeto

Este projeto consiste no desenvolvimento de um mini-site institucional para a empresa fictícia **Madeserra Madeireira**, que atua na venda de MDF, compensados, portas e acessórios para construção e marcenaria.

O objetivo foi criar uma presença digital simples, moderna e responsiva, contendo navegação entre páginas, layout organizado, formulário de contato e publicação online.

---

## 🏗 Estrutura do Projeto

O site é composto por três páginas principais:

- `index.html` → Página inicial (Home)
- `sobre.html` → Informações institucionais
- `contato.html` → Formulário e canais de contato
- `style.css` → Arquivo de estilização
- `README.md` → Documentação do projeto

---

## 🏠 Página Home

Contém:

- Título principal
- Texto introdutório
- Botão de chamada para ação (Solicitar Orçamento)
- Seção de produtos principais:
  - MDF
  - Compensados
  - Portas
- Seção de produtos complementares:
  - Ferragens
  - Maçanetas
  - Colas e fixadores
- Informações de atendimento regional

## 🏢 Página Sobre

## 📞 Página Contato

## 📱 Responsividade

O site foi desenvolvido utilizando Flexbox e Media Queries para garantir que os cards de produtos e seções institucionais sejam empilhados corretamente em telas menores (até 768px), garantindo boa experiência em dispositivos móveis.

---

## 🚀 Versionamento

O projeto foi versionado utilizando Git, com commits organizados conforme etapas:

1. Estrutura inicial HTML
2. Implementação de menu e rodapé
3. Estilização com CSS
4. Ajustes finais e responsividade
